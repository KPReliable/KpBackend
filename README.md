# KpReliable Backend

Simple Express + TypeScript backend scaffold.

## Available scripts

- `npm run lint` — run ESLint on `src`
- `npm run build` — run lint + TypeScript compile (`tsc`)
- `npm run dev` — run with nodemon on `index.ts`
- `npm run serve` — run with `node index.ts`
- `npm start` — run with `node app.ts`

## Quick start

1. `npm install`
2. `cp .env.example .env`
3. `npm run dev`

## Environment

- `PORT` default: 3000
